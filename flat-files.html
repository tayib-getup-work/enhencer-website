Flat Files

Flat files are terms to describe a group of data formats. The group includes; Excel Files (.xlsx), Comma Delimited(.csv) and Text Files (.txt). If your data is in one these formats please choose the flat files from the list in design screen.

This is one example of Excel (xlsx & .csv) file. This file represents customer records of a company, very specifically which customers have become churn or not. While preparing the data for uploading or analyzing please make sue your data meets the following conditions;

  1. There can not be a blank in the column name. Please understand that there can be missing values in the data bur specifically there can not me missing/blank value in the column name only.
  2. The characters used in the data should be Latin only, meaning all the other characters sets from other languages will have to be changed. For instance, Öö, Ğğ or similar different characters can cause problems in the analysis phase.
  3. Lastly, please make sure there is no blank space at the very bottom of the file and at the rightest side of the last column.
