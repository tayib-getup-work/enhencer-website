Enhence

Enhence Task

Enhence is where all the magic happens. It allows users to run machine learning algorithms with a click of a button and obtain powerful and actionable insights in matter of seconds.




On the right panel click the “Click To Enhence” button to start enhancing your data. Always remember the green highlighted variable on the left panel will be used as your target variable for the Enhence task meaning, all the models will be considering the selected variable as the target variable (aka: Dependent variable, Y).


Selecting Variables

In order to obtain models for the target variable user need to select other variables to see their relations and effects on the target variable (aka: Independent variable, Xi’s ).





Upon clicking the Enhence Button this screen will appear which allows user to select the desired variables for the model.

  1. These are all the variables present in the data to choose from.
  2. The button “All” can be used to select all the variable at once.
  3. The button “None” can be used to deselect all the selected variables.
  4. After selecting desired variables click the thick button to run the Enhence task and in few seconds the model will be ready.

Model Outputs

Once the model is created these buttons can be used to obtain the results of the model.




  • Select: Select can be used to to reselect and update model variables.
  • Tree: One of the results/output is Tree, can be used to see the hierarchical view of the data.
  • Segments: Segments can be used to obtain segments from automatically partitioned by the model.
  • Effects Effects can be used to obtain the variable importance from the model.
  • Options Options can be used to tweak the model settings.
