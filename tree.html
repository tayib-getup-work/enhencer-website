Tree

Tree can be used to obtain hierarchical view of the data according to the model.




  1. Once model is done running click the “Tree” button to obtain this output.
  2. On the right side of the analyze screen tree output can be seen. It’s a hierarchical view and here Relations and effects of other variables with the target variable can be observed.

Once the output displayed on the right-side users can observe the relations in details. All these bars can be called Nodes and all the bars with the leaf icon in front can be called leaf of the tree. Users can select any nodes to observe the statistics of the selected node. For instance,




  1. This node is currently selected node.
  2. Notice how the middle panels color is the same with the selected node, this is because the middle panel is showing the summary statistics of the selected node from the tree.
