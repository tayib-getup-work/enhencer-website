Analyze Screen GUI





This is the analysis screen of Enhencer. This is where all the analysis take place. You can access this page once the data upload/import is completed successfully. The GUI of analysis screen is broken down and described below;

  1. Left Panel in the analyze screen contains all the variables from the data.
  2. The green highlighted variable indicated the particular variable is selected as Target Variable for the analyzes. You can simply change the target variable by simply clicking a different variable.
  3. The Home button can be used to return to the workspace page where all the projects are stored.
  4. The Question button can be used to access the help center.
  5. This dropdown will bring down all the options regarding the user profile and account.
  6. The Predict button can be used to proceed to the prediction page once a model is being created for the data.
  7. Report button can be used to go the reports page where all the saved analyses are stored.

In the middle panel, you can see some statistics visualized on the selected variable whereas, Enhencer (prediction) results of selected segment, tree view leaf, etc. from the right panel.
