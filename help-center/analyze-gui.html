Analyze Screen GUI





This is the analysis screen of Enhencer. This is where all the analysis takes place. You will be automatically directed to this page once the data upload/import is completed successfully. The GUI of the analysis screen is described below;

  1. Left Panel in the analyze screen contains all the variables from the data.
  2. The green highlighted variable is the particular variable chosen as Target Variable for the analyses. You can change the target variable by simply clicking on a different variable.
  3. The Home button can be used to return to the workspace page where all the projects are stored.
  4. The Question mark icon can be used to access the help center.
  5. This dropdown will bring down all the options regarding the user profile and account.
  6. Predict button can be used to proceed to the prediction page once a model is created and saved for the data.
  7. Report button goes to the reports page where all the saved analysis reports are stored.

In the middle panel, you can see averages and distributions visualized on the selected variable and you can perform Filter and Group By actions. For Machine Learning to take place, click Enhence on the right panel.
