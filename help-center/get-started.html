Accessing Enhencer

Enhencer platform is a cloud based application but it can also be installed in-house. It is a web based application so you can access it and use it through your browser. The suggested browser to be used to access Enhencer is Google Chrome. Depending on the hosting of your system, you can access the system using the URL.

  1. Open Google Chrome.
  2. In the address bar, write the Enhencer application URL. If you are using Enhencer on cloud, the URL is https://app.enhencer.com If you are using Enhencer on your private cloud or in premise installation, the URL would be provided by your IT.

Accessing Enhencer is actually that easy, also you can access the app from the official wbsite enhencer.com and sign in from there.
