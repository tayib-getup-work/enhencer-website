Reports

In order to use the Report function of Enhencer user must conduct some analysis or model and save this. You can learn how to save an analysis or model here.

After saving an analysis or model simply click the report button at the top of the page and you will be redirected to the reports page.




The report page should look something like this. Left panel holds all the reports the user saved over time. One very important feature about reports is that the user can not change the results of any analysis or mode, user can only view them in different visualization options available.

  1. Green highlighted one is currently being selected for viewing purpose.
  2. Notice how many tasks like filter, group by, select etc. are missing. This is because Report page only allows users to view the results and doesnâ€™t allow to make any changes to the analyses.
