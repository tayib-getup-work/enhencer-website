Flat Files

Flat files are terms to describe a group of data formats. The group includes; Comma Delimited (.csv) and Text Files (.txt). If your data is in one these formats please choose the flat files option from the list in the design screen.

This is one example of Excel (xlsx & .csv) file. This file represents customer records of a company, specifically for a subscription-based product, which customers have left the company (churn) or not (retention). While preparing the data for uploading or analyzing, please make sure your data meets the following conditions;

  1. Column header cannot be blank. Please note that there can be missing values in the data but specifically, column name cannot be missing/blank.
  2. The characters used in the data should be Latin only, meaning all the other characters sets from other languages will have to be changed. For instance, Öö, Ğğ or similar different characters can cause problems in the analysis phase.
  3. Lastly, please make sure there is no blank space at the very bottom of the file and at the rightest side of the last column.
