Projects

Projects can be described as one or multiple predictive analytics models derived from a single historical dataset. In other words, it’s the analyses made and stored for one particular dataset.




The projects are stored under a specific workspace. For instance, these are the projects under the workspace “Product Analytics”.


Creating a New Project

In order to create a new project under the designated workspace, there are two ways, described below.




  1. On the right edge of the workspace bar simply click the+ button and a popup will appear to enter a name for the project. Simply provide a name and clickok to create a project.
  2. On the right edge of the workspace bar, click the… button that will bring down a dropdown menu. From the dropdown menu, select create project, enter a name and hit ok to create a project.

Project Options

Once a project is being created you can obtain the options available for the projects.





On the edge of the project, click the … to bring down the dropdown menu containing all the options for a project.

  1. Connect/Import: To upload or import a dataset please select this option.
  2. Reports: Reports can be used to go to the saved reports for the selected project.
  3. Delete: To delete the selected project select this option.
  4. Rename: This option can be used to rename a project.
