Workspace

Workspace GUI

This is the workspace of Enhencer. This is where all the projects are stored and displayed. To be more precise;




  1. Workspace: You can think of it as a folder for projects in Enhencer. Users can create multiple workspaces and collapse / expand them for better management of Projects.
  2. Project: One or multiple predictive analytics models derived from a single historical dataset.

Creating a New Workspace

To create a new workspace simply click the button + Create Workspace and a popup will appear asking for the name of the workspace. Provide the desired name and click Ok to finalize creating a new workspace.



Workspace Options

If you click on the â€¦ button at the corner of the workspace tab, an option will pop up. These are the options for the selected workspace.





  • Create Project: This can be used to create a new project under the selected workspace.
  • Delete: Click delete to delete the selected workspace and all of its projects.
  • Rename: Rename can be used to simply rename the selected workspace.
  • Add Collaborator: This can be used for granting access to another user for the current workspace. Collaborators can view the projects but cannot edit them.
  • Remove Collaborator: This can be used to remove other users from accessing the selected workspace.
