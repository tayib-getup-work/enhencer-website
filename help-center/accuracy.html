Accuracy

Accuracy tab brings the accuracy graph for the selected model. Accuracy is one of the most important performance metrics for a predictive model. It helps users to compare models and select the most accurate and best one.




  1. Click the “Accuracy” tab to obtain the accuracy graph for the selected model.
  2. The large dots represent the proportion of data that is chosen to observe accuracy for the model. You can increase/decrease by clicking Plus and Minus signs above the graph to see the changes happen when selecting a different proportion of data.

Threshold: Threshold value is the classification probability for the choices.
Overall Error: Overall error represents the model error rate for the whole population.
Type 1 Error is False Positive: The percentage of new data marked as Target choice positive but in fact, they will be target choice negative.
Type 2 Error is False Negative: The percentage of new data marked as Target choice negative but in fact, they will be target choice positive.
