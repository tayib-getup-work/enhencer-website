Enhence Function

Enhence function allows users to run machine learning algorithms with a click of a button and obtain powerful and actionable insights in a matter of seconds.




On the right panel, hit Click To Enhence button to start enhancing your data. Always remember that the green highlighted variable on the left panel will be used as your target variable for the Enhence task. All the models you build will be considering the selected variable as the target variable (aka: Dependent variable, Y).


Selecting Variables

In order to obtain models for the target variable, the user needs to select other variables to see their relations and effects on the target variable (a.k.a. Independent variable, Predictor variable, Xi’s ).





Upon clicking the Enhence Button this screen will appear which allows the user to select the desired independent variables for the model.

  1. These are all the variables present in the data to choose from.
  2. All button can be used to select all the variable at once.
  3. None button can be used to deselect all the selected variables.
  4. After selecting desired variables, hit "✓" to run the Enhence task, and in a moment the model will be ready.

Model Outputs

Once the ML model is built, these buttons can be used to view/adjust the results of the model.




  • Select: Select can be used to change and update the independent variables of the model.
  • Tree: One of the results/output is Tree, which shows the hierarchical view of the data.
  • Segments: Segments shows the segments automatically grouped by the model.
  • Effects Lists the independent variables of the model, from most to least affecting.
  • Options Options can be used to tweak the model settings.
