Options

For diving deeper, Options opens up advanced settings of the Enhance function. The enthusiast users can experiment with the default settings of the algorithms, naturally expecting different results. Moreover, if the tree is unable to split with the default settings then one should change the settings from the Options.




  1. Confidence Level is the probability that the value of a selected variable falls within a specified range of values.
  2. This refers to the Minimum leaf size which is determined by default setting. However, the user can easily change the minimum segment size for the tree and segments functions. Doing so will update the tree and the user might obtain a different result than before.
  3. Maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node. In other words, this is the depth/height of the tree, this should neither be too small nor too large. The provided values are determined by the default setting which the user can change as they see fit.
