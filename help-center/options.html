Options

For the enthusiasts “Options” brings advanced settings for the “Enhance” task. The enthusiast users can experiment with the settings expecting different results. Moreover, If the tree refuses to split with the default settings then one can change the settings from the Options.




  1. “Confidence Level” is the probability that the value of a selected variable falls within a specified range of values.
  2. This refers to the "Minimum leaf size" which is determined by default setting. However, user can easily change the minimum segment size for the tree and segments functions. Doing so will update the tree and the user might obtain different result than before.
  3. "Maximum depth" is the number of nodes along the longest path from the root node down to the farthest leaf node. In other words, this is the depth/height of the tree, this should neither be too small nor too large. The provided values are determined by the default setting which the user can change as the users sees fit.
