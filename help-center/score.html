Scoring New Data

When it comes to obtaining prediction score of new data Score or View task can be used.




You can score new data;

  1. Select “Choose File” button under New Data on the Right Panel. Select the file containing new data to be scored. Click Open.

When new data is uploaded to Enhencer, Enhencer compares it with the base historic data and displays the results in Variable Comparison Screen.




This is what it looks like when new data upload is done for prediction;

  1. Left column of variables are significant variables in the model
  2. Middle column of variables are matched variables in the new data
  3. Right column of variables are unmatched variables in the new data

Even if the model data structure and new data structure are same (recommended) the unmatched variables in new data column will be filled with target variable and insignificant variables.
All unmatching fields in new data will also be displayed here.
Users need to match significant variables to new data. This means each significant variable need to have a parallel new data variable.
Sometimes the names can have slight differences like a space is missing or a character is extra or any kind of typo errors. In such cases just drag and drop the unmatched variables from the right side to the left side and it will be done. Once everything is set click “Apply” to start scoring the new data.

Scoring and Exporting the Scores

On the Left Panel under New Data, a message indicating the scoring algorithm is running will appear with a progress bar.
Once the scoring algorithm run ends, there will be a new message saying “Prediction scores for the data are ready” as well as an Export function.

Click the “Export All” button to export the prediction score of the new data.
