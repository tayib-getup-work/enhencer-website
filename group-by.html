Group By

In order to observe the relation of a variable on the target variable Group By task can be used. In other words user can see the realtion among two variables with group by.




A drop down menu will appear after clicking Group By. The drop-down menu offers all the variables from the data that the user can select to observe the group by effects. Users can select any varaible they want.
After a variable is choosen output will look a lot like this;




  1. The variable name inside the group by box represents the variable selected for the group by task.
  2. The Target Choice drop-down menu provides the choices form the target variable. If a target choice is selected than the respective target choice will be highlighted in the diagram shown in the middle panel.
  3. The Number in the right corner represents the sample size of the selected segment. Segment size shows how many individual records fall into that segment.
  4. The right panel shows the choices for the variable selected for the group by task.

Segments are sorted from highest lift to the lowest lift therefore you can scroll down for many other segments and the bottom segment has the lowest lift among all the segments.


Interpretation:

Why do we need a group by and what is it ain the first place?

Lets try to figure out does the age of the customers effect in any way in becoming churn. In such a case our target variable is the Customer Status. Also remember that customer status has two distinct values, Churn & Retention meaning a customer either become churn or not. Thus lets choose Churn from the “Target Choice” dropdown.

As the group by variable lets choose “Age” from the drop down menu. You will see age is devided into 3 groups automatically by Ehencer. Enhencer does this to find the most optimal thresholds.

On the left you will see a stacked bar chart that displays the group by results.

Notice the overall churn proportion in the data is 25%. It seems customer aging between 52-68 are most likely to become churn compared to other age groups.

This is how group by helps user to see how one particular variable can effect their target variable.
