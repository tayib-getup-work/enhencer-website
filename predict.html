Predict

Predict screen is the place where you can score data using saved models, select different machine learning algorithms and export suitable predictive models to external platforms (currently Qlik script and SQL are supported).




This is the Predict Screen of Enhencer, you can understand in which page and task you are by looking at the highlighted bar in the top section which is currently set on Predict. These are the basic explanations of the GUI;

  1. In Predict Screen for a project, you can save multiple models. These models will appear in the Left Panel of Predict screen.
  3. Green highlighted one is currently being selected for prediction purpose.
  4. The right panel gives the user to implement and see the effect of different Machine Learning Modelling algorithms and compare their result for the best possible prediction possible.
  5. The Middle panel shows the results of the chosen model and the model specification like life, accuracy, scoring etc.
  6. On the left: These are the model output options like accuracy, life and likelihood. On the Right: These are model options like updating model with different parameter, scoring new predictions and many more.

All these are explained in detailed manner in next segments.

Model Algorithms


On the right panel Enhencer provides the different Machine Learning Modelling algorithms that users can choose from. Currently 0nly Decision Tree is available to implement, however, all the other model algorithms will be ready to use very soon.


Also, users can tweak the primary settings of the model algorithms from the right side. For instance, under Decision Tree there is Minimum Leaf Size and Maximum Depth options that can be changed to see how the model will perform for predictions. Just play with the settings and choose the best options that defines the data best.
